About

This project aims to develop a comprehensive management support system tailored for hospital administration. The system is designed to enhance decision-making processes, improve operational efficiency, and optimize resource allocation within a hospital setting.

Features

Nexira offers several features to assist decision-makers in their process. It includes tools like control charts for data analysis, a simulation experience for outpatient departments, and the ability to assess and compare multiple hospitals. These features help improve decision-making and overall hospital management.

    ControlCharts

    Nexira uses control charts to monitor your data and determine if everything is running smoothly. We then provide you with a detailed report, including analysis observations and easy-to-understand graphs that illustrate all the control charts used.

    Outpatinet DepartmentAnalysis

    Nexira offers a comprehensive analysis of the Outpatient Department using the uploaded data. Following this, it generates a detailed report containing insights, key points, and observations. Additionally, Nexira provides individual simulations for each clinic, accompanied by full reports that include all key points and analysis observations.

    Waiting Time ReductionSolutions

    Nexira offers two solutions to reduce patient waiting times in clinics: implementing a booking system and establishing a new clinic. It provides simulations for each solution, predicting and visualizing their impact on reducing waiting times.

    Hospitals Comparison and Assessment

    Nexira uses two methods to analyze hospital input/output data. First, the ordering method, which primarily relies on normalization, ranks hospitals based on their mean and consistency (standard deviation). Next, Data Envelopment Analysis (DEA) is applied to determine each hospital's efficiency. Nexira then visualizes the rankings and highlights any inefficient hospitals, providing a summary report that clarifies all analysis insights.
